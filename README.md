## Objetivo del proyecto
Desarrollar un archivo de Excel que organice una base de datos de ventas, utilice herramientas de análisis apoyadas por IA o análisis predictivo, genere visualizaciones y responda una pregunta clave del negocio.

## Tema elegido
**Análisis de ventas mensuales con proyección del próximo trimestre**

## Pregunta clave
**¿En qué mes del próximo trimestre se proyectan mayores ventas y qué categoría lidera el acumulado?**

## Fuente de datos
Datos sintéticos creados con fines académicos.  
La base contiene **54 registros** y columnas con nombres claros.

## Herramientas usadas
- Excel: tablas, gráficos, fórmulas y proyección.
- Analizar datos / Ideas o gráficos recomendados de Excel.
- IA complementaria para estructurar el problema, definir la pregunta clave y organizar la documentación.

## Hojas del Excel
1. **Portada**: resumen general del proyecto.
2. **Datos**: tabla base de ventas.
3. **Analisis**: KPIs y dos visualizaciones principales.
4. **Pronostico**: proyección del próximo trimestre.
5. **Documentacion**: tema, fuente, herramientas y conclusiones.
6. **Guia_IA**: pasos para mostrar el uso de IA durante el video.

## Principales hallazgos
- La categoría con mayor participación en el acumulado es **Tecnología**.
- El mes con mayor venta real del histórico es **diciembre de 2024**.
- La proyección lineal muestra que el mes más alto del próximo trimestre sería **septiembre de 2025**.

## Cómo presentar el proyecto en el video
1. Mostrar la hoja **Datos**.
2. Explicar cómo usaste **Analizar datos / Ideas** o los gráficos recomendados.
3. Mostrar la hoja **Analisis** con las visualizaciones.
4. Mostrar la hoja **Pronostico** y responder la pregunta clave.
5. Cerrar en **Documentacion** con las conclusiones.
